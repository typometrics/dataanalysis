MAL Effect — log-log regression slopes β(1→max)

Pegah Faghiri, Kim Gerdes, Sylvain Kahane (2026). Verifying the Menzerath-Altmann law in the verbal domain in 180 languages. UDW26 @ LREC 2026.

Statistical threshold: a value MAL_n is included in the regression only if it has at least 100 occurrences. Mini-plots use a single global axis range so slopes are visually comparable.

Effect categories: β > 0.1: MAL · β < -0.1: Anti-MAL · |β| ≤ 0.1: gray zone.

Languages above the diagonal show stronger RMAL than LMAL (RMAL bias); languages in the bottom-right quadrant are the RMAL but Anti-LMAL family that drives the asymmetry discussed in §5.1 ("we can speak of both an RMAL bias and an Anti-LMAL bias").

Distribution of β(1→max) across languages — MAL / LMAL / RMAL

Interpretation: Each bar counts how many languages have a β(1→max) in that bin for the given direction. Bars to the right of zero show MAL compliance; bars to the left show anti-MAL behavior. Bin width is 0.05.
