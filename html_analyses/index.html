📊 Menzerath-Altmann Law (MAL) Analysis Reports

Interactive HTML reports analyzing the Menzerath-Altmann Law across languages from Universal Dependencies treebanks. Each report includes tables, charts, and visualizations showing how constituent size varies with the number of dependents.

📁 Available Reports

  • Complete MAL Analysis All Languages
    Full analysis of all languages with bilateral (total), left-only, and right-only dependent sections. Includes sortable tables, MAL Effect scores, local change scores, and interactive charts.
  • VO Languages Analysis VO Score > 66%
    Analysis of head-initial/VO languages where the verb typically precedes its object. Includes directional breakdown (left/right dependents).
  • OV Languages Analysis VO Score < 33%
    Analysis of head-final/OV languages where the verb typically follows its object. Includes directional breakdown (left/right dependents).
  • Mixed Word Order Languages VO Score 33-66%
    Analysis of languages with mixed or flexible word order patterns. Includes directional breakdown (left/right dependents).
  • 📓 Notebook Plots & Commentary
    All images and markdown commentary from the MAL analysis notebook. Includes β scores, asymmetry analysis, decay rates, trajectories, and universality tests.
  • 🗺️ UD Language Maps
    Interactive world maps showing all Universal Dependencies languages. Map 1: Languages colored by family. Map 2: Languages sized by corpus size.

📖 About the Analysis

Menzerath-Altmann Law (MAL) states that as a linguistic construct increases in size (number of constituents), the average size of its constituents decreases. In this analysis:

  • MAL_n: Arithmetic mean size of dependents for verbs with n total dependents
  • MAL Effect Score: Measures overall MAL compliance (positive = compliant)
  • Local Change Score: Elasticity measure between consecutive n values

Data is extracted from Universal Dependencies treebanks (version 2.17), analyzing verbal constructions across 180+ languages.
