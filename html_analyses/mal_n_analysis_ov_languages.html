📊 Total (Bilateral) ⬅️ Pure Left ➡️ Pure Right

MAL_n Analysis: OV Languages (VO Score < 33%)

Head-final/OV languages where the verb typically follows its object.

Data source: Bilateral dependency configurations from Universal Dependencies treebanks

Minimum occurrences: 10 per configuration (to ensure statistical reliability)

MAL_n: Geometric mean of constituent sizes for heads with n total dependents

MAL Effect Score: Mean of all local change scores for a language (positive = MAL compliance)

Local Change Score (n→n+1): [ln(MAL_n) - ln(MAL_{n+1})] / [ln(n+1) - ln(n)]

• Positive values (green): MAL compliance - constituent size decreases as n increases

• Negative values (red): Anti-MAL - constituent size increases with n

• Values near 0 (yellow): Weak or no effect

Visualizations

1. Mean MAL_n Curve Across Languages

Interpretation: This chart shows the average MAL_n trajectory across all languages. The shaded area represents the interquartile range (25th-75th percentile), showing cross-linguistic variation. A downward trend confirms MAL: as n increases, constituent size decreases on average.

Note: The number of contributing languages decreases at higher n values (see Data Availability chart), which may affect the reliability of the mean at those points.

2. Normalized MAL Curve (MAL_n / MAL_1)

Interpretation: This chart shows the relative change in constituent size, normalized by each language's starting point (MAL_1 = 1.0 for all languages). This allows direct comparison of the rate of decline across languages, independent of their initial constituent sizes.

Key insight: All languages start at 1.0; the steeper the decline, the stronger the MAL effect. A value of 0.8 at n=3 means constituents are 80% of their size at n=1.

3. Data Availability by Number of Dependents

Interpretation: This chart shows how many languages have sufficient data (≥10 occurrences) at each value of n. The sharp decline at higher n values explains why mean curves may become less reliable and potentially show unexpected patterns (survivor bias).

4. Distribution of Local Change Scores by Transition

Interpretation: True box plot showing the distribution of local change scores across languages for each transition. The box spans the interquartile range (Q1 to Q3), with the median marked. Whiskers extend to the most extreme points within 1.5×IQR, and outliers are shown as individual dots.

Color coding: Green boxes indicate median > 0.1 (MAL compliant), yellow = near zero, red = median < -0.1 (anti-MAL).

Key insight: Values above the dashed zero line indicate MAL compliance — constituent size decreases as n increases.

5. Individual Language MAL_n Trajectories

Interpretation: Each line represents one language's MAL_n trajectory. The thick black line shows the cross-linguistic mean. Most languages show a downward trend (MAL compliance), but there is considerable variation in both the starting point (MAL_1) and the slope of decline.

Interactive: Hover over lines to see language names.

6. Histogram of All Local Change Scores

Interpretation: This histogram shows the overall distribution of local change scores across all languages and transitions. A distribution centered above zero indicates general MAL compliance. The spread shows how consistent the MAL effect is across the dataset.

7. MAL_1 vs. Average Slope (Initial Size vs. Decline Rate)

Interpretation: Each point represents one language. The x-axis shows MAL_1 (initial constituent size when there's only 1 dependent), and the y-axis shows the mean local change score (average slope of the MAL curve).

Key questions:

  • Is there a relationship between starting size and rate of decline?
  • Do languages with larger initial constituents show stronger or weaker MAL effects?
  • A negative correlation would suggest a ceiling effect; a positive correlation would suggest compound effects.

Color coding: Points are colored by language family. Hover for details.

8. MAL Effect Score by Language Family

Interpretation: The MAL effect score is the mean of all local change scores for a language. Positive values indicate MAL compliance (constituent size decreases with n); negative values indicate anti-MAL behavior.

Family comparison: This chart groups languages by family to reveal whether MAL strength varies systematically across genealogical groups. Box plots show the distribution within each family.

Key insight: If MAL is a universal constraint, we expect positive effect scores across all families. Systematic differences might indicate structural factors that modulate MAL strength.

9. World Map: Geographic Distribution of MAL Effect

Interpretation: Each dot represents a language positioned at its geographic location. The color indicates the global MAL effect score (mean of all local change scores for that language).

Color scale: Green = strong MAL compliance (constituent size decreases with n), Yellow = weak/neutral effect, Red = anti-MAL (constituent size increases with n).

Geographic patterns: If MAL is truly universal, we expect green dots distributed across all continents. Clustering of colors might suggest areal effects or shared structural features.

Note: Only languages with geographic coordinates available in WALS are shown.
